# Offer of a Lifetime

Прототип 2D карточной игры на Unity. Игрок собирает персонажа через карты предыстории, затем разыгрывает карты действий чтобы заработать достаточно денег и получить оффер мечты.

## Структура проекта

```
offer_of_a_lifetime/
├── offer_of_a_lifetime_client/   # Unity-проект (игра)
└── auto_tests/                   # Папка для авто-тестов (пока пустая)
```

## Как играть

**Фаза 1 — Предыстория:**
Открываешь 4 карты рубашкой вниз. Каждая определяет одну характеристику персонажа:
- Возраст и бэкграунд
- Базовый навык (C#, Manual QA и т.д.)
- Стартовые ресурсы (деньги + дни)
- Статус занятости

После каждой карты появляется попап с описанием и бонусами. После закрытия 4-го попапа начинается игра.

**Фаза 2 — Игра:**
В центре экрана карты действий. Каждая стоит деньги и дни, бросается d20 против порога карты.
- Успех → получаешь награду (`rewardMoney`)
- Провал → ресурсы потрачены зря
- Нет ресурсов → попап-предупреждение

Карты предыстории переезжают в бар снизу экрана — всегда видны.

**Конец игры:**
- Победа: деньги ≥ 1000$ → «Оффер получен!»
- Поражение: дни ≤ 0 → «Время вышло...»

Итоговый экран показывает деньги, количество попыток и успехов. Кнопка «Сыграть ещё» перезапускает сцену.
